# Kata: Password Check

You will build a password check app with HTML, CSS and JavaScript.

## Features

- [ ] Add these five checks
  - [x] Passwords are equal
  - [x] Password contains at least one lower case character
  - [x] Password contains at least one upper case character
  - [x] Password contains at least one number
  - [x] Passwort is at least 10 characters long
- [x] The first check is always the "equality check" of both passwords
  - [x] If this first check fails all other checks will be failed automatically
- [x] Run all checks whenever the user types into any of the password inputs
- [x] Add a button that toggles the text in the password fields and the label of the button itself
  - [x] If the passwords are hidden the text of the button is "Show Passwords"
  - [x] If the passwords are visible the text of the button is "Hide Passwords"

## Styling

- [x] Style the app as visible on the screenshots and in the video
- [x] Use `lightsalmon` as background color
- [x] Use [Lobster](https://fonts.google.com/specimen/Lobster) for Headings and [Open Sans](https://fonts.google.com/specimen/Open+Sans) for all other text
- [x] Use Open Sans also for inputs and buttons
- [x] Use a one column layout below 700px screen width
- [x] Use a two column layout above 700px screen width
- [x] Use emojis (either from your emoji keyboard or [emojicopy](https://www.emojicopy.com/)) for valid or invalid checks
- [x] Inputs and buttons should have
  - [x], A black border
  - [x] A 2px border radius

## Hints

Input elements for passwords are created like this.

```html
<input type="password" />
```

Input elements for text are implemented like this.

```html
<input type="text" />
```
